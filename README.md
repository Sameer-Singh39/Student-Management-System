# Student JSON Data Project

## Overview
This project contains student information stored in JSON format. JSON (JavaScript Object Notation) is a lightweight data-interchange format that is easy for humans to read and write and easy for programs to parse and generate.

The file stores details of students such as:
- Student ID
- Student Name
- Age
- Marks
